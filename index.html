Breng me thuis

Breng me thuis is an location aware app designed to function as a Dutch Train travel guide. It calculates the fastest route to the closest station and from that station to your destination. This is done using the NS api and open street map. In this project we imlpemented the MVVM principles. It will pull train information from the NS Rest API and use it to make a travel guide to the desired destination (has to be NS station). The app is written in java with a friend of mine. Doctor fitness test

Fitness test is an windows application made in c#. This application is made to calculate the fitness of clients for a doctor. It was designed so a doctor can communicate with clients and see their progress. This is all done through a BLE connection with a bike and heartrate sensor. This project was build with an client server architecture. The client can use VR glasses to visualize the test in a virtual world. The VR processing was done using an extern server to whom JSON packets were send to build the world and handle the camera position and bike. In this project was C# .net core and wpf used. The project was made with 5 other students. For the source code click here.
